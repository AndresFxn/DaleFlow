═══════════════════════════════════════════════════════════════════
                    ⚠️ IMPORTANTE - LEE ESTO PRIMERO
═══════════════════════════════════════════════════════════════════

Si ves errores como "vite no se reconoce" o "artisan no se reconoce":

1. Ejecuta: verificar-requisitos.bat
2. Instala lo que falte (PHP, Composer, Node.js)
3. Ejecuta: install.bat
4. Ejecuta: start.bat

Ver SOLUCION-RAPIDA.md para ayuda detallada.

═══════════════════════════════════════════════════════════════════
                        🚀 INICIO RÁPIDO
═══════════════════════════════════════════════════════════════════

1. Ejecuta: verificar-requisitos.bat (verificar instalaciones)
2. Ejecuta: install.bat (instalar dependencias)
3. Configura PostgreSQL en backend\.env
4. Ejecuta migraciones: cd backend && php artisan migrate
5. Ejecuta: start.bat (iniciar proyecto)

DISEÑO:

  • Estilo femenino y elegante
  • 100% responsive

TECNOLOGÍAS:
  • Backend: Laravel 10
